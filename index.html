Propriedades
text-decoration

    Essa pripriedade adiciona ou remove linhas no nosso texto: seja linhas em cima, no meio, ou embaixo.

  • text-decoration-none
  • text-decoration-line-through
  • text-decoration-underline
  • text-decoration-overline
    •

text-decoration-none;

Ele exclui as atribuições setadas originalmente.

Exemplo A:
Neste exemplo usando o código sem extilização CSS o link ficará com um "traçado" típico no rodapé.

( Link não Estilizado ) código HTML 5
< a href="#"> < /a>

Exemplo B:
Neste exemplo usando o código com extilização CSS o link ficará sem o "traçado" típico no rodapé.

( Link Estilizado ) Código HTML 5
< a class="exemplo-b" href="#"> < /a>

Código CSS usando a class="exemplo-b"
.exemplo-b{
text-decoration: none;
}

text-decoration-line-through;

Ele irá adicionar um traçado ao meio do texto.

Exemplo:

Seu texto irá ficar assim

Código em CSS

.line-through{
text-decoration: line-through;
}

text-decoration-underline;

Ele irá adicionar um traçado ao meio do texto.

Exemplo:

Seu texto irá ficar assim

Código em CSS

.underline{
text-decoration: underline;
}

text-decoration-overline;

Ele irá adicionar um traçado ao meio do texto.

Exemplo:

Seu texto irá ficar assim

Código em CSS

.overline{
text-decoration: line-through;
}

Propriedades
text-decoration-line

    Essa pripriedade adiciona ou remove linhas no nosso texto: seja linhas em cima, no meio, ou embaixo.

  • text-decoration-line;
    1. text-decoration-line: line-through;

      2. .line-01{
      text-decoration-line: line-through;
      }
      Usando a classe: line-01
    2. text-decoration-line: underline;

      3. .line-01{
      text-decoration-line: underline;
      }
      Usando a classe: line-02
    3. text-decoration-line: overline;

      4. .line-03{
      text-decoration-line: overline;
      }
      Usando a classe: line-03
  • text-decoration-style;

      • Será usando para melhor didatica o comando:
      text-decoration-line: underline; para poder ver a funcionalidade do comando text-decoration-style: e suas propriedades.

    1. text-decoration-style: solid; (Default)

      2. .style-01{
      text-decoration-line: underline;
      text-decoration-style: solid;
      }

      Como padrão ele cria uma linha continua no texto.

    2. text-decoration-style: double;

      3. .style-02{
      text-decoration-line: underline;
      text-decoration-style: double;
      }

      Ele cria uma linha dupla em todo o texto.

    3. text-decoration-style: dotted;

      4. .style-03{
      text-decoration-line: underline;
      text-decoration-style: dotted;
      }

      Ele cria uma linha pontilhada ao longo do texto.

    4. text-decoration-style: dashed;

      5. .style-04{
      text-decoration-line: underline;
      text-decoration-style: dashed;
      }

      Ele cria uma linha tracionada ao longo do texto.

    5. text-decoration-style: wavy;

      6. .style-05{
      text-decoration-line: underline;
      text-decoration-style: wavy;
      }

      Ele cria uma linha com ondas ao longo do texto.
  • text-decoration-color;
    1. text-decoration-color: (cor da linha);

      2. .color-01{
      color: rgb(254, 254, 255);
      text-decoration-line: underline;
      text-decoration-color: darkblue;
      }

      Ele cria uma linha indepedente da cor do texto.

  • text-decoration-thickness;
    1. text-decoration-thickness: 5px;

      2. .thickness-01{
      text-decoration-line: underline;
      text-decoration-thickness: 5px;
      }

      Ele aumenta a espeçura da linha em px, ou outra unidade de medida.
